# Task app
  To do app using Ruby on Rails
  This software is under construction

## Development Environment 
- ruby 2.2.3p173
- Bundler 1.10.6

## Install and Run
```
$ bundle install --path vendor/bundle  
$ bin/rake db:migrate RAILS_ENV=development  
$ bundle exec rails server  
```
